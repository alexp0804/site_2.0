Maze Generator

This is one of my favorite uses of a backtracking algorithm because you can literally see it doing the backtracking. The end result is a nice maze!

It works by picking a random square that it can go to while pushing the square it's leaving onto a stack. Once it runs into a dead end, it pops the last square off the stack until it runs out of options again.

I like to think of it as a randomized depth-first-search, which it technically is.
